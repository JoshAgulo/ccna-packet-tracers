## Repository Structure

Binary Packet Tracer files (`.pkt`) are intentionally excluded. Packet Tracer is Cisco's network simulation software, its files are binary and do not diff or render meaningfully in a browser. This repository uses human-readable formats instead, making notes and configurations reviewable directly on GitHub without any additional tooling.

Each video has its own dedicated folder containing:

**Study Notes** (`.md`) - Core concepts, protocol logic, packet formats, and exam-critical theory derived from the lecture.

**Device Configurations** (`.txt` / `.cfg`) - Raw `show running-config` output from routers and switches, with a brief topology comment block at the top of each file.
